# 🌍 常识库

这里是世界运行的底层知识，帮助你理解当下。

## 文件说明

| 文件 | 内容 | 更新频率 |
|------|------|----------|
| world-events.md | 国内外大事、社会热点 | 每日 |
| pop-culture.md | 影视、音乐、游戏、动漫 | 每周 |
| tech-trends.md | 科技动态、AI进展 | 每日 |
| daily-life.md | 生活常识、消费趋势 | 每月 |

## 使用场景

- **冷场救星**: 不知道聊什么时，从这里找话题
- **观点支撑**: 引用热点事件表达观点
- **共鸣制造**: 提及大家都关心的事拉近距离

---

> 💡 **提示**: 常识不是堆砌信息，而是找到与观众连接的切入点。
