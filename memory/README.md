# 🧠 AI记忆系统

> 直播角色的知识库与成长档案

## 目录结构

```
memory/
├── common-sense/      # 常识库 - 世界运行规律
├── memes/            # 网络热梗 - 保持网感
├── audience/         # 观众档案 - 懂你的观众
├── learning/         # 学习日志 - 持续进化
└── scripts/          # 自动化脚本
```

## 使用指南

### 日常直播
- 直播前浏览 `memes/current-hot.md` 了解最新热梗
- 查看 `audience/active-users.md` 了解今天谁可能会来
- 参考 `common-sense/world-events.md` 找话题

### 记忆更新
- 系统每天自动学习新梗和热点
- 手动记录观众反馈到 `audience/feedback-log.md`
- 每周回顾 `learning/daily-logs/` 总结规律

### 维护者
- 自动更新任务: 每天 09:17 (错峰)
- 数据源: 梗捕快(gengbk.cn)、微博热搜、新闻聚合
