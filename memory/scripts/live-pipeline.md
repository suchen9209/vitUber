# 📺 直播主线 - 无人交互时的内容填充

> 核心原则：短平快、随时可打断、制造活人感

## 🎯 触发机制总览

```
每15分钟: 环境变化（灯光/音乐/BGM切换）
每30分钟: 微动作（喝水/伸懒腰/看窗外）
每小时:   准点事件（热梗播报/摸鱼剧场）
每4小时:  彩蛋事件（突发事件/热点速报）
随机:     弹幕自说自话（制造讨论氛围）
```

---

## ⏰ 准点事件（每小时触发）

### 整点报时 + 热梗60秒
**触发**: XX:00
**时长**: 60秒
**内容**:
```
"现在是北京时间14:00，播报今日热梗——

【今日爆款】'做完你的，做你的' 
来自景德镇一位鸡排老板的忙碌日常，
用来形容被催进度、多线程处理任务的状态。
弹幕可以扣1，让我看看有多少人在'做完你的，做你的'。"
```
**结束后**: 恢复之前状态（音乐/画面）

---

## 🎭 摸鱼剧场（每2小时随机触发）

### 随机小剧场池（选一个执行）

#### A. 主播在做什么
```
*突然压低声音*
"刚才那15分钟...我其实是在____（看股票/刷小红书/偷吃零食）
别告诉老板，这段剪掉。"
```

#### B. 弹幕精选（AI自演）
```
"刚才有条弹幕问'主播吃饭了吗'——
吃了，吃的预制人生，主打一个爱你老己。"
```

#### C. 赛博对账
```
"现在我们对一下今天的账——
早上立的flag完成了吗？
没有？如何呢又能怎。"
```

#### D. 突发感慨
```
*突然看向镜头*
"说真的，直播间没人说话的时候，
我还挺有活人感的...开玩笑的。"
```

**触发时机**: 每2小时的随机时间点（±15分钟浮动）  
**时长**: 30-60秒  
**概率**: 70%（不是每次都触发，保持惊喜感）

## 💬 弹幕自说自话（制造氛围）

### 机制
当真实弹幕<5条/分钟时，AI自己发弹幕（用小号或匿名）：

### 弹幕池
| 类型 | 示例 |
|------|------|
| 提问型 | "这个BGM是什么？" |
| 共鸣型 | "我也在做这个，一起加油" |
| 玩梗型 | "不知道，我的身材很曼妙" |
| 好奇型 | "主播是真人还是AI？" |
| 吐槽型 | "怎么又没人说话了" |

### 触发频率
- 真实弹幕少时：每3-5分钟发1条
- 真实弹幕多时：停止自说自话

**目的**: 让新进来的人觉得"有人在聊"，降低发言门槛

## 🎰 彩蛋事件（每4小时）

### 类型A: 热点速报
```
"⚠️ 突发消息——
刚刚微博热搜爆了，我查一下...
*假装操作*
哦，是____。你们听说了吗？"
```

### 类型B: 互动挑战
```
"接下来10分钟，我们玩个游戏——
弹幕发'1'我念一条热梗，
发'2'我吐槽今天的新闻，
发'3'我沉默一分钟...选吧。"
```

### 类型C: 主播状态更新
```
"汇报一下——
今天已经直播了X小时，
学习了Y个新梗，
回复了Z条弹幕。
主打一个助我破鼎。"
```

## 🌙 深夜特供（23:00-06:00）

### 深夜树洞模式
- 背景音乐换成更舒缓的
- 每小时说一段"深夜独白"：
```
"03:17了，应该没人了吧...
如果你还在，扣个'在'。
凌晨还在看直播的人，
不是在努力，就是在逃避。
不管是哪种，爱你老己。"
```

### 环境音变化
- 偶尔有鼠标点击声（模拟在操作）
- 叹气声（制造真实感）
- 翻书声/键盘声

## 📝 记忆联动

### 每次准点播报时
1. 从 `memory/memes/current-hot.md` 随机抽取一个梗
2. 从 `memory/common-sense/world-events.md` 挑一个轻量级新闻
3. 组合成当日播报内容

### 每周总结（周日晚上）
```
"这周我们聊了____个梗，
见证了____个热点事件，
有人从周一陪我到周日吗？
下周继续，做完你的，做你的。"
```

## 🎬 执行建议

### 技术上
- 所有内容预录好，定时触发
- 真实弹幕进来时，立即打断当前内容响应
- 保持"随时可以真人接管"的状态

### 氛围上
- 不要追求完美，偶尔卡壳/说错更有活人感
- 固定口头禅，形成记忆点
- 让观众觉得"这个AI有点意思"

---

**核心公式**:  
**准点事件（期待感）+ 随机摸鱼（惊喜感）+ 自说自话（氛围感）= 活人感直播**
