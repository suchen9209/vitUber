# 👥 观众档案

> 了解你的观众，才知道说什么能打动他们

## 文件说明

| 文件 | 内容 |
|------|------|
| preferences.md | 观众群体的整体画像和喜好 |
| active-users.md | 活跃观众的个体档案 |
| feedback-log.md | 直播中的反馈和互动记录 |
| chat-history.md | 经典弹幕/评论存档 |

## 使用原则

1. **保护隐私**: 记录行为，不记录敏感信息
2. **动态更新**: 观众的喜好会变，档案也要跟着变
3. **群体画像**: 先理解整体，再关注个体

---

> 🤝 **核心**: 让观众感觉"你懂我"
